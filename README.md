## Képernyők

| Tab | Tartalom |
|-----|----------|
| 🎯 **Bingó** | Welcome hero + 5×5 interaktív játéktábla, progress bar, konfetti |
| 👥 **Közösség** | Top-5 leaderboard, napi kihívások, élő badge |
| 📰 **Hírek** | Szatirikus editorial timeline feed, reakciókkal |
| ⚙️ **Beállítások** | Profil stats, értesítések toggle, 2FA, kijelentkezés |

## Tech stack

| Réteg | Technológia |
|-------|-------------|
| UI | React 18 |
| Stílus | Tailwind CSS 3 (Material Design 3 színrendszer) |
| Ikonok | Google Material Symbols |
| Betűtípusok | Plus Jakarta Sans + Manrope |
| Build | Vite 5 |
| Deploy | Vercel |

## Helyi futtatás

```bash
npm install
npm run dev
```

## Kapcsolat a régi verzióval

Ez a projekt **független** a `valasztasibingo` repótól. Párhuzamosan üzemelnek:
- `valasztasibingo` → egyszerű bingó app
- `kozeletimozaik` → teljes 4 képernyős app (ez a repó)
